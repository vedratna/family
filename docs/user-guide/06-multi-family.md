# Multi-Family

> One account, multiple families. Switch between them seamlessly.

## How It Works

A single FamilyApp account can belong to any number of families. For example:

```
Priya's Account
 │
 ├── Sharma Family (in-laws)  [Teal]
 └── Verma Family (parents)   [Coral]
```

Each family is a completely separate space with its own:

- Feed (posts, photos, comments)
- Calendar (events, reminders)
- Family tree (members, relationships)
- Chores
- Theme color
- Member list and roles

**No data crosses between families.** Posts in Sharma Family are invisible to Verma Family and vice versa.

## Family Switcher

Tap the **family name** in the header to open the switcher. Each family shows its accent color as a visual indicator — you can tell which family you're in at a glance.

```
┌─────────────────────┐
│  Your Families       │
│                      │
│  ██ Sharma Family  ◀── active (Teal)
│  ██ Verma Family      (Coral)
│                      │
│  + Create Family     │
└─────────────────────┘
```

When you switch, the entire app updates — feed, calendar, tree, chores, and header color all change to the selected family's context.

## Creating Additional Families

Any member can create a new family at any time:

1. Open the family switcher
2. Tap **+ Create Family**
3. Name it, pick a theme color, invite members

You become the Owner of the new family. Your existing families are unaffected.

## Roles Across Families

Your role can be different in each family. For example:

- **Owner** of Sharma Family (you created it)
- **Editor** in Verma Family (your parent created it)

Permissions apply per-family based on your role in that specific family.
